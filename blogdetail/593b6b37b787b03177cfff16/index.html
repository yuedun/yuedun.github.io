mongoose自动添加createdAt和updatedAt

月盾
2017-06-10 04:44
MongoDd
目录
export var CommentSchema: Schema = new Schema({
    Id: String,
    content: String,
    status: {
        type: Boolean,
        default: true
    }
}, { timestamps: true});

添加{ timestamps: true}选项即可，如果需要自定义属性名称可以直接指定： { timestamps: {createdAt:'created_at', updatedAt:'updated_at'}}
